# URL Existence Checker

A lightweight, client-side application built with Vanilla TypeScript that validates user-input URLs and asynchronously checks their existence via a mock server.

> **Note:** This project is currently under active development.

## Setup

To run this project locally, ensure you have [Node.js](https://nodejs.org/) installed. Then, follow these steps in your terminal:

**1. Install dependencies:**
   ```bash
   npm install
   ```

**2. Compile the TypeScript code:**
   ```bash
   npx tsc
   ```

**3. Start the local server:**
   ```bash
   npx serve .
   ```

## Testing

**To run tests, execute:**
   ```bash
   npm run test
   ```
